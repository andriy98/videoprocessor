## Description

Simple Nest JS application with Bull Jobs queue

Listens to the 'video' queue and processes it.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

NOTE: make sure Redis installed and running on your local machine.


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
